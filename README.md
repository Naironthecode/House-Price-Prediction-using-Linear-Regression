This project focuses on predicting house prices using Linear Regression, a fundamental machine learning algorithm. It utilizes features such as area, number of bedrooms, and bathrooms to estimate property prices.

The workflow includes data loading, preprocessing, and cleaning to handle missing values and select relevant features. Exploratory Data Analysis (EDA) is performed to understand relationships between variables. The dataset is then split into training and testing sets, and a Linear Regression model is trained on the data.

Model performance is evaluated using metrics like Mean Squared Error (MSE) and R² score, providing insights into prediction accuracy. Visualizations are used to compare actual and predicted values.

Overall, this project demonstrates a complete machine learning pipeline and provides practical experience with Python libraries such as Pandas and Scikit-learn.
